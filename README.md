# Dimpho Coffee Shop Sales Analysis
# Overview
This report analyses the daily transactional dataset from Dimpho Coffee Shop. 
The aim is to support the new CEO in increasing revenue and optimizing product performance.
The focus is on extracting actionable insights from historical data and provide a set of 
recommendations 
to support the  CEO in decision-making.
# Objectives
’āś	Identify top revenue-generating products. 
’āś	Determine peak sales times during the day. 
’āś	Analyse sales trends across products and time intervals. 
’āś	Recommendations for improving sales performance.
#  Tools and Technologies
’āś	Databricks: Data storage, transformation, and SQL analysis
’āś	Excel: Pivot tables, charts, and visual analysis
’āś	PowerPoint: CEO presentation
 # Miro: Data flow and architecture planning

# Data Processing as Summarised:
’āś	Received Excel data, converted to CSV, and loaded into Databricks. 
’āś	Performed data cleaning and transformations, including correcting entries and computing total amounts. 
’āś	Created 7 new tables from the transformed data. 
’āś	Used SQL to aggregate data by category, time bucket, and spend bucket for analysis.
Exported transformed data back to Excel.
